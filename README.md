# 🚨 Guia de Segurança Digital para Idosos

Este é um projeto de **Landing Page Educativa** focada em acessibilidade para o público da terceira idade. O objetivo é ensinar, de forma simples e interativa, como identificar e se prevenir de golpes comuns no WhatsApp e no Pix.

## 🌟 Diferenciais do Projeto
Este projeto foi desenvolvido com foco em Idosos, aplicando:
- **Alto Contraste:** Cores fortes para facilitar a leitura.
- **Tipografia Adaptada:** Fontes grandes e legíveis.
- **HTML5 Semântico:** Estrutura organizada para leitores de tela.
- **Interatividade:** Quiz, simulador de mensagens e checklist de segurança.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura semântica.
- **CSS3:** Estilização responsiva e acessível.
- **JavaScript:** Lógica para o quiz, simulador e contador de segurança.

## 🚀 Funcionalidades
- **5 Regras de Ouro:** Dicas rápidas de proteção.
- **Detetive Digital:** Simulador que destaca palavras suspeitas em mensagens.
- **Desafio de Segurança:** Quiz interativo com feedback imediato.
- **Escudo Digital:** Checklist que calcula o nível de proteção do usuário.
- **Fontes Oficiais:** Links diretos para órgãos de segurança (Febraban, Banco Central).
